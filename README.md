# PITOL QR PRO

Professional QR Code Generator and Scanner Progressive Web Application.

## Features

### QR Generator

- Text QR
- URL QR
- WiFi QR
- Email QR
- Phone QR
- vCard QR
- Custom colors
- Logo embedding
- PNG export
- SVG export
- PDF export


### QR Scanner

- Camera scanner
- Image scanner
- Scan history
- Quick actions


### Batch Generator

- CSV import
- Multiple QR generation
- ZIP export


### Application

- Responsive design
- Dark/light theme
- Offline support
- Installable PWA
- Local storage
