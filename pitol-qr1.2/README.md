# PITOL QR PRO


A professional QR Code Generator and Scanner Progressive Web Application.


## Features


### QR Generator

- Text QR
- URL QR
- WiFi QR
- Email QR
- Phone QR
- vCard QR
- Custom colors
- Logo support
- PNG export
- SVG export
- PDF export


### QR Scanner

- Camera scanner
- Image scanner
- Scan result display


### Batch Generator

- CSV import
- Multiple QR generation
- ZIP export


### Application

- Responsive design
- Dark mode
- Light mode
- Offline support
- Installable PWA
- LocalStorage history



## Technology

- HTML5
- CSS3
- JavaScript ES Modules
- PWA API
- LocalStorage
